# Writing a Screencast Video Editor in Haskell

A talk for Lambda World Cadiz 2018

## Building

This requires:

* Make
* TexLive
* Pandoc 2.x
* pandoc-include-code
* pandoc-emphasize-code >= 0.2.4

Run:

```
make
```
